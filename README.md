# Discord Bots Dashboard

لوحة تحكم متكاملة لإدارة البوتات على Discord باستخدام React و Node.js و Firebase.

## الهيكل

- **frontend/**: واجهة المستخدم React
- **backend/**: السيرفر و الـ API
- **services/firebase.js**: الربط مع Firebase
- **.env**: إعدادات البيئة
- **package.json**: Dependencies و Scripts
